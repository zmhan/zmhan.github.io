Hugo部署到Github问题记录

04 Jan 2018, 06:03
hugo / GitHub

在将Hugo部署到GitHub中时，遇到了一系列问题，大多问题都很快解决了。以下这些问题，有些是在英文搜索中发现解决的，有些是自己摸索着解决的，记录下来，备忘。

更新：以下两个问题在Hugo官方网站都可找到对应解决办法。因此：1、多看官方guide和documents；2、对于此类问题的解决方案，英文资料更加丰富。

  1. 本地预览正常，但部署到GitHub后，CSS失效。

    • 原因：baseurl错误。

    • 解决方式：与网站地址完全一致。

  2. 绑定个人域名，在网站添加cname，再次提交更新时会被删除。

    • 原因：每次生成的public文件夹下没有cname文件。
    • 解决方案：在themes»theme»static文件夹中新建一个cname文件，这样再次生成public时就会有cname了。
